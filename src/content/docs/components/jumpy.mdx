## API Reference
### Root

| Prop           | Default                                                   | Type                            | Description                                                  |
|:---------------|:----------------------------------------------------------|:--------------------------------|-------------------------------------------------------------|
| `images`       | []                                                         | `Image[]`                       | Array of image objects with `src`, `alt` and `href` (optional) properties      |
| `minRotation`  | 7                                                         | `number`                        | Minimum rotation angle for images in degrees                |
| `maxRotation`  | 8                                                         | `number`                        | Maximum rotation angle for images in degrees                |
| `wrapperStyles`| "w-20 h-20 rounded-xl ring-[4px] ring-white overflow-hidden" | `string`                     | Additional classes for the image wrapper                    |
| `imageStyles`  | ""                                                        | `string`                        | Additional classes for the image                            |
| `gap`          | "-ml-5"                                                   | `string`                        | Gap between images (as a Tailwind class)                    |
| `separation`   | 40                                                        | `number`                        | Separation distance when hovering over images               |
| `initEase`     | "elastic.out(0.5, 0.3)"                                   | `gsap.EaseFunction \| string`   | GSAP easing function for initial animation                  |
| `hoverEase`    | "power4.out"                                              | `gsap.EaseFunction \| string`   | GSAP easing function for hover animation                    |
| `duration`     | 0.5                                                       | `number`                        | Duration of the animations in seconds                       |
